Edge %
Edge % measures how much a player's recent average exceeds (or falls short of) the prop line.

Formula: (Avg − Line) / Line × 100

+5% means the player averages 5% above the line. Positive = lean Over.
Confidence
Confidence (0–100) is a composite score rating how much we trust the edge signal.

Components:
• Edge magnitude → up to +20
• Consistency (low variance) → up to +20
• Hit rate extremity → up to +3
• DvP matchup quality → −5 to +10
• Recent trend alignment → ±5
• Base score: 50
Hit Rate
Hit Rate = % of the last 10 games where the player cleared the prop line.

80% = cleared in 8 of 10 games.
50% = coin flip. Above 60% is encouraging.
DvP
Defense vs Position (DvP) ranks how easy each opposing defense is to score against, for a player's position.

#1 = easiest matchup (defense allows most pts to that position)
#30 = hardest matchup

Easy = rank 1–10 · Mid = 11–20 · Hard = 21–30
Value Score
Value Score combines edge, hit rate, and confidence into one number for quick ranking.

Formula: Edge×0.35 + HitRate×0.30 + Confidence×0.35

Scores ≥ 30 with edge > 4% earn the ⚡ VALUE badge.
Scenario
Scenario Mode estimates how a player's production might increase when a key teammate is listed as out.

Star teammate missing: +3–5 pt boost estimate
Role player missing: +1–2 pt boost estimate

Based on historical usage redistribution patterns.
